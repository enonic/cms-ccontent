Source server - use userstore\username if not in default userstore
Target server - use userstore\username if not in default userstore

[1]Sourceserver URL is the remote server (f.x. http://localhost:8082/rpc/bin) rpc bin endpoint which you want to copy content FROM, cms.security.rpc.enabled=true must be set in cms.properties for this server, and firewall has to allow access between this and that server.

[2]Targetserer URL is the target server (f.x. http://localhost:8080/rpc/bin) rpc bin endpoint which you want to copy content TO, cms.security.rpc.enabled=true must be set in cms.properties for this server, and firewall has to allow access between this and that server.
